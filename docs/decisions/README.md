# Архитектурные решения

ADR фиксируют решения, которые сложно восстановить из кода. Номер не переиспользуется. Принятый документ не переписывается задним числом; изменение оформляется новым ADR со ссылкой на заменяемое решение.

Статусы: `proposed`, `accepted`, `deprecated`, `superseded`.
