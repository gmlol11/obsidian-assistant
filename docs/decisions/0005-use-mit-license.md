# ADR-0005: Использовать открытую лицензию MIT

- Статус: accepted
- Дата: 2026-07-16

## Контекст

Проект планируется публиковать на GitHub. Без явной лицензии код можно просматривать, но права на использование, изменение и распространение остаются ограниченными авторским правом.

## Решение

Опубликовать проект под лицензией MIT с указанием правообладателя `gmlol11` и года первой публикации 2026.

## Последствия

- код можно свободно использовать, изменять и распространять при сохранении уведомления об авторстве и лицензии;
- программное обеспечение предоставляется без гарантий;
- лицензия не содержит отдельного явного патентного положения, характерного для Apache-2.0;
- новые исходные файлы репозитория распространяются на тех же условиях, если явно не указано иное.
