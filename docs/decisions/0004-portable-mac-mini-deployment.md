# ADR-0004: Готовить переносимое развёртывание на Mac mini

- Статус: accepted
- Дата: 2026-07-16

## Контекст

Разработка начинается на текущем Mac, но целевая среда — домашний Mac mini с постоянным аптаймом. Прямого доступа к нему на старте нет.

## Решение

Код, конфигурация и пользовательские данные разделяются. Путь к vault и интеграционные ключи передаются окружением. Репозиторий содержит контейнерный профиль, документацию macOS и smoke tests, но не содержит реального vault или секретов.

## Последствия

- локальная разработка использует фикстуру;
- перенос не требует изменения исходного кода;
- системные настройки macOS выполняются отдельно при появлении доступа;
- контейнер не решает FileVault, вход пользователя, питание и backup — они остаются частью runbook.
