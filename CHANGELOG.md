# Журнал изменений

Формат основан на Keep a Changelog. Версионирование проекта следует Semantic Versioning после первого публичного релиза.

## [Unreleased]

### Added

- первоначальная структура публичного репозитория;
- документация продукта, архитектуры, безопасности и эксплуатации;
- ADR для ключевых архитектурных решений;
- безопасный CLI для диагностики и создания входящей заметки;
- тестовое хранилище и автоматические проверки;
- контейнерный профиль с отключённой сетью и минимальными правами;
- открытая лицензия MIT;
- версионированный контракт входящего события `capture.text`;
- локальная файловая очередь вне vault с состояниями pending, processing, completed и quarantine;
- идемпотентность по `request_id` и fingerprint события;
- повторные попытки, processing lease, восстановление после сбоя и ручной retry;
- CLI-команды `queue enqueue`, `status`, `process`, `retry` и `recover`;

### Security

- `dry-run` включён по умолчанию;
- запись ограничена allowlist папок;
- перезапись существующих файлов запрещена;
- секреты исключены из Git;
- завершённые квитанции не хранят payload или actor ID;
- runtime запрещено размещать внутри vault или над ним;
- восстановление после уже выполненной записи принимает только полностью идентичный файл.
